# 🐍 Snake Game

A classic Snake Game built with **Python** using the **Turtle** graphics library. This project follows an **Object-Oriented Programming (OOP)** approach by separating the game logic into multiple modules.

---

## ✨ Features

- 🐍 Smooth snake movement
- 🍎 Random food spawning
- 📈 Real-time score tracking
- 💥 Collision detection
- 🎮 Game Over screen
- 🔄 Object-Oriented design

---

## 🛠️ Technologies Used

- Python
- Turtle Graphics
- Object-Oriented Programming (OOP)

---

## 📂 Project Structure

```
Snake-Game/
│
├── main.py          # Main game loop
├── snake.py         # Snake class
├── food.py          # Food class
├── score.py         # Scoreboard class
└── README.md
```

---

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/MohamedZidan13/Snake-Game.git
```

2. Open the project folder.

3. Run the game:

```bash
python main.py
```
